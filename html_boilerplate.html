Entities

To represent reserved characters in the text of an html page, we have to use entities.

If you look at the source code for the following list, you will see that the space and the brackets are creating by some weird encoding.

  • this bracket <
  • this other bracket >
  • and  the spaces  between   these r   words
  • the "And" symbol &

They all start with the symbol "And" followed by a semicolon at the end. Inbetween these two is the entity code.
